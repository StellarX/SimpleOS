项目说明：   一个32位的操作系统

z_tools文件夹：项目工具

os文件夹内：（主要项目文件）
ipl10.nas：启动程序加载器（将OS从磁盘读入内存）
Makefile：包括文件生成规则和命令
make.bat：调用make.exe(这个内部的命令不懂)，而make.exe的作用是进入makefile文件，然后执行里面的命令
run.bat：运行qemu pc模拟器
haribote.nas：存放操作系统
